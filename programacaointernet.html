Estruturando páginas com HTML

Juntamente com a linguagem de estilo Cascading Style Sheets (CSS) e a linguagem de programação JavaScript, a Hypertext Markup Language (HTML) é uma das três principais linguagens importantes para o desenvolvimento frontend.

Versões HTML

HTML: a primeira versão do html foi publicada em 1992 e foi proposta por Tim Berners-Lee.

HTML 4.0: 1997 e foi muito importante para o CSS.

XHTML 1.0: reformulação do HTMl, segundo as regras do XMl.

HTML5: Lançada em 2014 e substitui as versões antigas de HTML e XML.

Uso de elementos e atributos

Como já mencionado o HTML é uma linguagem de marcação que serve para estuturar páginas web, bem como sua semântica. Cada elemento vai consistir em uma abertura de tag e fechamento de tag. Por exemplo, as tags "h1 e /h1" intruduzem o titulo inicial, dando destaque ao seu tamanho e indicandio assim que é um título principal.

Quando utilizamos o Doctype estamos especificando a versão do html que iremos utilizar.

Quando abrimos os <HTML> estamos introduzindo a página web, além disso as informações inseridas dentro dessa tag são aquelas que não serão exibidas no "corpo" do nosso site.

Já quando fazemos uso de <BODY> etsmos falando que tudo que está sendo inserido ali deve aparecer no browser para ser visto, já que ali é o "corpo" do site.

E quando falamos de texto podemos utilizar as tags que indicam titulos, subtitulos, seus tamanhos, etc. São as tags <h1> essa tag indica os títulos principais da página e pode se ramificar em mais cinco tags indicando textos menores, ou seja, vai até o h6, sendo “h” a abreviação de “heading” (título).

Parametrização de elementos via atributos

Os atributos são colocados na tag de abertura de um elemento e consistem em um nome e um valor que fica entre aspas duplas, separados por um sinal de igualadade. Exemplo, para definir um link utilizamos a tag <a> que é uma ancora e usamos o atributo <href> para definir a URL da página vinculada.

Uso dos titulos, paragrafos e outras formatações de etexto.

Como já mencionado, o HTMl possui seis tipos de headings. Cada uma delas vai corresponder a seis diferentes elementos, sendo a primeira para o primeiro nível e maior e o ultimo, para o menor. Para definir paragrafos usamos a tag <p> Os parágrafos são exibidos pelo navegador com um espaço entre eles e outros parágrafos. <i> e <b>: o primeiro deixa o texto em itálico (“italic”), e o segundo em negrito (“bold”). Você também pode usar os elementos <em> e <strong> para destacar textos importantes. Por padrão, textos com <em> aparecem em itálico, e textos com <strong> aparecem em negrito. Alguns elementos podem também ser úteis como, por exemplo, <blockquote> que serve para citações maiores.

Criação de listas

Existem três tipos de listas diferentes em HTML: as listas ordenadas (enumeradas), as listas não ordenadas (não enumerdadas) e listas de definição (efinição de termos).

  1. First entry
  2. Second entry
  3. Third entry
  4. Fourth entry
  5. Fifth entry

Para listas não ordenadas usa-se <ul> que basicamente significa "unordered list"

Links

Links são criados com a tag <a> e o atributo href. Podem ser externos, relativos ou internos.
O texto do link deve ser claro. Os links são divididos nos links externos, ou seja, que te redirecionam para outras páginas ou os links internos que te redirecionam para páginas dentro do próprio site web.

Incluindo Imagens

Imagens usam a tag <img>. Atributos principais: src (caminho) e alt (descrição).Podem ter legenda com <figure> e <figcaption>.

Dados em Tabelas

As tabelas são usadas para organizar dados em linhas e colunas.Elementos principais:
- <table>: tabela.
- <tr>: linha.
- <td>: célula.
- <th>: cabeçalho.
Para melhor organização, pode-se usar <thead>, <tbody> e <tfoot>. É possível mesclar células com colspan (colunas) e rowspan (linhas).

Formulários

Formulários permitem coletar dados dos usuários. O elemento principal é <form>, que possui:
- action: para onde os dados são enviados.
- method: forma de envio (GET ou POST).
Campos comuns:
- <input> (texto, senha, email, etc.)
- <textarea> (textos longos)
- <select> e <option> (listas)
- radio (uma opção) e checkbox (várias opções)
- submit (botão de envio)
Os dados são enviados como pares nome=valor.
